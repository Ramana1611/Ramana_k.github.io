About Web Technology

Web technology refers to the multitude of technologies and tools that are utilized in the process of communication between different types of devices over the internet. It enables the delivery of web content and the creation of interactive web applications.

Key Components

  • HTML (HyperText Markup Language): The standard markup language for creating web pages.
  • CSS (Cascading Style Sheets): Used for describing the presentation of a document written in HTML.
  • JavaScript: A programming language that enables interactive web pages.
  • PHP: A server-side scripting language designed for web development.
  • MySQL: A relational database management system used with web applications.
  • Perl: A highly capable, feature-rich programming language with over 30 years of development.

Importance of Web Technologies

In today's digital age, web technologies form the backbone of all websites and web applications. They enable businesses to reach global audiences, provide platforms for e-commerce, facilitate communication, and offer services that were unimaginable just a few decades ago.

Did You Know?

The first website ever created was by Tim Berners-Lee in 1991. It explained what the World Wide Web was and how to use it.

HTML Theory

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Download Theory Notes

Key concepts:

  • HTML documents are made up of elements that define the structure of a web page
  • Elements are represented by tags like <html>, <head>, <body>
  • Attributes provide additional information about elements
  • HTML5 introduced semantic elements like <header>, <footer>, <article>

Sample HTML Code:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

CSS Theory

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Key concepts:

  • Separation of content (HTML) from presentation (CSS)
  • Selectors target HTML elements for styling
  • The box model: margin, border, padding, content
  • Responsive design principles

Sample CSS Code:

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    color: #333;
}

.container {
    width: 80%;
    margin: 0 auto;
}

.button {
    padding: 10px 15px;
    background-color: #3498db;
    color: white;
    border: none;
    border-radius: 4px;
}

Perl Theory

Perl is a family of high-level, general-purpose, interpreted, dynamic programming languages. Perl was originally developed by Larry Wall in 1987 as a general-purpose Unix scripting language to make report processing easier.

Key features:

  • Powerful text processing capabilities
  • Supports both procedural and object-oriented programming
  • CPAN - Comprehensive Perl Archive Network with thousands of modules
  • Regular expression support built into the syntax

Sample Perl Code:

#!/usr/bin/perl
use strict;
use warnings;

print "Content-type: text/html\n\n";
print "<html><body>";
print "<h1>Hello, World!</h1>";
print "</body></html>";

my $name = "Ramana";
print "Hello, $name!\n";

PHP Theory

PHP is a popular general-purpose scripting language that is especially suited to web development. It was originally created by Rasmus Lerdorf in 1994 and is now maintained by The PHP Group.

Key features:

  • Server-side scripting language
  • Can be embedded directly into HTML
  • Supports object-oriented programming
  • Extensive database support
  • Large standard library

Sample PHP Code:

<!DOCTYPE html>
<html>
<body>

<?php
echo "<h1>Hello, World!</h1>";

$name = "Ramana";
echo "Welcome, " . $name . "!";
?>

</body>
</html>

MySQL Theory

MySQL is an open-source relational database management system (RDBMS). Its name is a combination of "My", the name of co-founder Michael Widenius's daughter, and "SQL", the abbreviation for Structured Query Language.

Key concepts:

  • Relational database model
  • Tables, rows, columns
  • Primary keys, foreign keys
  • SQL queries: SELECT, INSERT, UPDATE, DELETE
  • Indexes for performance optimization

Sample SQL Code:

-- Create a database
CREATE DATABASE web_tech;

-- Create a table
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    roll_no VARCHAR(20) UNIQUE,
    email VARCHAR(100) UNIQUE
);

-- Insert data
INSERT INTO students (name, roll_no, email)
VALUES ('Ramana', 'E22ASL304', 'ramana@example.com');

-- Query data
SELECT * FROM students WHERE roll_no = 'E22ASL304';
